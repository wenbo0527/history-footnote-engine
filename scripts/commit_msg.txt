v2.10.1 W70: LLM 失败时 state 自动回滚（避免白扣 AP）

问题：
- game._run_round() 内部扣 AP → LLM 失败 → 异常抛出
- 之前 except 块只返 500 → state 仍被修改
- 结果：玩家看不到结果但 AP 已扣 + 跳月 + 变量已变

修复（input.py:166+）：
1. try 前 _state_snapshot = {...} 快照
   - action_points_current/max
   - current_date + round_number
   - narrative_history_len
   - variables + value_shifts（dict 浅拷贝）
2. except 块内 _rollback_state 还原
   - action_points_current/max
   - current_date + round_number
   - narrative_history 截断到失败前长度
   - variables / value_shifts 还原

3. 错误消息更新：suggestion 含"行动点已退还，状态已回滚"

测试：
- 10 个 W70 单元测试 PASS（含 AP/date/round/narrative/variables 多场景）
- 0 回归

影响：
- LLM 失败 → state 完全不变 + 返 500
- 玩家重试同 input → 行为一致
- 不再"白扣 AP"
