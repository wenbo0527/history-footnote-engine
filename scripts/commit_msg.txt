v2.10.1 W80.1: 修复 useFateCard import 错误（API 名实为 fateUse）

- SyntaxError: The requested module '/src/lib/api/fate.ts'
  does not provide an export named 'useFateCard'

- import { useFateCard } from '$lib/api/fate'  // ❌ 不存在
- 实际 fate.ts export 的是 fateUse

修复：
- import { fateUse } from '$lib/api/fate'
- handleUse 调 fateUse(sessionId, cardId, context)
- context 映射：card.use_type → context
  round_start → round_start
  emergency → emergency
  其他 → immediate
