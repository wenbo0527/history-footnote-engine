## 支线C：织工路线——“从机户到机工”

### 核心定位

不是“选择”，而是“被选择”——当经营失败、卖掉织机，你就从老板变成了雇工。这是最自然的下坠路径，也是万历年间大量织户的真实命运。

### 前置规则：什么情况下会变成机工

| 触发条件 | 具体表现 | DM判断标准 |
|---------|---------|-----------|
| 丝价暴跌 | 生丝价格下跌，织出的绸卖不上价 | 连续2个行情周期亏损 |
| 牙行压价 | 丝绸牙行联合压价，收购价低于成本 | 玩家与牙行关系差 |
| 水灾/天灾 | 丝料受损，无法按时交货 | 触发季节事件 |
| 高利贷到期 | 借钱买丝料，到期还不上 | 玩家此前借过钱 |
| 加税 | 孙隆税卡加征（1587年后逐渐升级） | 触发时代事件 |

**关键规则**：从机户变成机工不是一瞬间的事，而是一个逐渐滑落的过程——先减织机，再借高利贷，最后卖掉最后一台织机。

### 阶段拆解

#### 阶段一：挣扎期（还能撑）

- **状态**：还有1-2台织机，但入不敷出
- **选择**：借高利贷续命 / 减少雇工自己多织 / 找牙行求情
- **代价**：高利贷月息约3-5分（年息30-60%），越借越深
- **概率**：约60%的挣扎期机户最终会滑落到阶段二

#### 阶段二：卖机期（最后一台织机）

- **状态**：卖掉最后一台织机还债，变成无产者
- **选择**：去丝市等活当机工 / 转行做其他手艺 / 离开盛泽
- **代价**：卖织机所得约5-10两（视新旧），但可能还不够还债
- **转折**：这是“留在丝织业”还是“彻底转行”的分水岭

#### 阶段三：机工期（丝市等活）

- **日常**：每天天亮去丝市等活，牙人介绍工作抽成
- **收入**：日薪约3-4分银，月约1两——勉强糊口
- **住宿**：租住“机房”（大机户提供的集体宿舍）或寺庙偏房
- **风险**：没有固定雇主，“朝不谋夕，得业则生，失业则死”
- **社交**：每天和几百个机工挤在丝市，消息灵通

#### 阶段四：觉醒期（接触集体行动）

- **触发**：听到苏州城机工闹事的消息 / 税吏来催征 / 牙人压价太狠
- **选择**：继续忍 / 跟着去请愿 / 加入集体行动
- **代价**：参与集体行动可能被捕、被处死
- **终极事件**：万历二十九年（1599）葛成起义——见支线E

### 机工日常细节

**丝市等活流程**：
1. 天亮前到丝市（盛泽镇中心）
2. 牙人（中介）站在高处喊活——“某号机房要织工X名”
3. 机工挤上去抢活，牙人抽成约10%
4. 没抢到活的只能明天再来

**收入与支出对比**：

| 项目 | 月收入/支出 | 说明 |
|------|-----------|------|
| 机工日薪 | 3-4分银/天 | 月约1两 |
| 伙食 | 约0.3两/月 | 粗茶淡饭 |
| 住宿 | 约0.2两/月 | 机房集体宿舍 |
| 家庭开支 | 约0.5两/月 | 妻子络丝贴补 |
| **结余** | **约0-0.1两** | 几乎没有余钱 |

### 女性差异

**女性机工的处境**：
- “妻络夫织”模式下，女性主要做络丝（将蚕丝整理上机），男性做织造
- 女性也可以去丝市等活，但工种偏“前道”（缫丝、络丝），工资比织造低约20%
- 如果丈夫还在，女性通常在家络丝，丈夫去丝市等活
- 如果丈夫不在（死亡/离乡），女性可以独立做络丝工

**女性机工的特殊风险**：
- 丝市环境复杂，女性容易被骚扰
- 怀孕期间无法工作，没有收入
- 丧夫后可能被迫改嫁或沦为佣工

### DM植入示例

**场景1：卖掉最后一台织机**
> 你站在空荡荡的作坊里，最后一台织机已经被搬走了。买主给了你六两银子，还完高利贷还剩不到一两。妻子在门口默默看着你，手里还攥着半匹没织完的绸。明天，你也得去丝市等活了。

**场景2：丝市等活**
> 天还没亮，丝市已经挤满了人。牙人老周站在台阶上喊：“三号机房要织工五名，手艺好的优先！”几十个人涌上去，你被挤到了后面。旁边一个老机工叹气：“今天又抢不到活了。”

**场景3：听到葛成的消息**
> 阿根从苏州城回来，带回一个消息：“昆山来的机工葛成，在玄妙观聚了几千人，要跟税监算账！”丝市上的人交头接耳，有人兴奋，有人害怕。你握紧了拳头——你也是被税卡逼到这条路上来的。

### 小说/历史素材

- **《醒世恒言·施润泽滩阙遇友》**：施润泽从穷到富的反向参照——如果施润泽运气不好，就会变成机工
- **曹时聘奏报**：“吴民生齿最繁，恒产绝少……浮食寄民，朝不谋夕，得业则生，失业则死”——最真实的机工写照
- **葛成事迹**：从普通机工到“葛将军”——织工路线的终极叙事

### 交叉互动

- **与支线A（读书人）**：识字机工可能被大商人聘为管账——转入入幕/师爷路线
- **与支线B（西门庆）**：机工可能被“打行”招募——转入灰色路线
- **与支线E（反抗）**：机工是反抗路线的主力——织工路线的终极出口
- **与离乡路线**：机工没有牵挂，最容易选择离乡

## 支线D：出家/山人路线——“看破红尘”

### 核心定位

精神层面的退出选择。万历年间江南佛教复兴是独特历史背景，但出家远没有想象中简单——度牒要钱，女性几乎不可能正式出家，寺庙也不是世外桃源。

### 前置规则：出家的硬约束

#### 男性出家

| 条件 | 具体要求 | 难度 |
|------|---------|------|
| 度牒 | 官府发的出家许可证，可以买卖 | 需数两银子（需确认具体价格） |
| 寺院接收 | 需要找到愿意接收的寺院 | 中等——大寺院门槛高，小寺院容易 |
| 考察期 | 先做沙弥，经僧团考察合格才能受戒 | 至少1年 |
| 戒律 | 不饮酒、不吃荤、不近女色 | 对普通人很难 |

**度牒获取方式**：
1. **官府发放**：每年定额发放，竞争激烈
2. **购买**：从已有度牒者手中购买，或从寺院购买空额度牒
3. **私度**：没有度牒偷偷出家——违法，被查到要还俗受罚

#### 女性出家——几乎不可能

| 限制 | 具体规定 | 来源 |
|------|---------|------|
| 年龄限制 | 未满59岁不许为尼 | 建文三年（1401）法令 |
| 违法后果 | 杖一百还俗，入官为奴 | 《大明律》 |
| 嘉靖补充 | 发还原籍出嫁，庵产入官 | 嘉靖六年法令 |
| 万历极端 | 按猪肉价格卖给光棍 | 万历三十三年法令 |

**女性替代路径**：
1. **居士**：在家修行，不受年龄限制，但社会地位低
2. **带发修行**：住在庵堂但不剃度，灰色地带
3. **山人**：不做宗教仪式，以诗文书画自娱——男女皆可

### 阶段拆解

#### 路径A：正式出家（男性）

**阶段一：买度牒**
- 费用：数两银子（需确认）——对织户来说是一大笔钱
- 来源：找寺院方丈帮忙 / 找牙人购买 / 找官府关系
- 风险：买到假度牒 / 被骗

**阶段二：投寺做沙弥**
- 选择寺院：大寺院（灵岩寺、寒山寺）门槛高但资源多；小寺院容易进但清苦
- 日常：念经、扫地、种菜、化缘
- 考察期：至少1年，方丈满意才能受戒

**阶段三：受戒成僧**
- 仪式：剃度、受戒（比丘戒253条）
- 身份：正式僧人，免除租税徭役
- 日常：念经、做法事、管理寺产

**寺庙经济的真相**：
- 大寺庙有大量田产，收租放贷——方丈更像“地主”
- 僧人不是“出世”的——要参与寺庙的经济管理
- 化缘是重要收入来源——和施主搞好关系很关键

#### 路径B：山人路线（男女皆可）

**核心定位**：不做宗教仪式，但隐居乡间，以才华谋生——“有文化的自由职业者”

**阶段一：做出决定**
- 触发：科举落榜 / 经营失败 / 对现实失望
- 标志性动作：陈继儒1587年“取儒衣冠焚弃之”——与时代包起始年完全吻合！
- 代价：放弃科举/经商的前途，社会地位下降

**阶段二：建立名声**
- 方式：写诗文 / 画画 / 编书 / 交游
- 关键：需要进入文人圈——有人引荐
- 收入：卖字画（一幅约0.5-2两）、编书刻书、打秋风

**阶段三：游走权贵**
- 方式：给官员当幕僚 / 给富商当清客 / 参加文人雅集
- 风险：被利用 / 被卷入党争 / 名声受损
- 收入：幕僚月入约2-5两，远高于机工

**山人的日常**：
- 上午：读书、写字、画画
- 下午：访友、参加雅集、谈生意
- 晚上：编书、写文章
- 收入不稳定，但社会关系广

### 女性差异

**女性出家**：59岁以下禁令是硬约束，几乎不可能正式出家

**女性山人**：
- 理论上可行——明代有女性知识人“积极走向社会，与文人士大夫唱和交游”
- 但需要依附于某个男性文人圈——父亲、丈夫、或赞助人
- 可做的事：编诗集、教富户女儿读书（闺塾师）、卖刺绣/字画
- 社会接受度远低于男性山人——会被议论“抛头露面”

**女性居士**：
- 在家修行，吃斋念佛
- 可以参加寺庙的法会、放生等活动
- 社会接受度高——“善女人”是正面形象
- 但没有经济独立性——仍需靠家庭供养

### DM植入示例

**场景1：考虑出家**
> 你站在灵岩寺的山门前，看着进进出出的僧人。一个知客僧走过来问：“施主可是要进香？”你犹豫了一下，说：“我想见方丈。”知客僧上下打量你：“出家？你有度牒吗？”

**场景2：陈眉公的启发**
> 你在镇上的茶馆里遇到一个中年人，穿着布衣却气度不凡。他正在给人写扇面，一笔好字。有人叫他“眉公先生”。你听说他二十九岁就烧了儒衣冠，再不考科举，靠编书写画过日子，竟然活得比很多当官的还自在。

**场景3：女性玩家的困境**
> 你去庵堂找净莲师太，问她能不能出家。师太叹了口气：“你才二十多岁，朝廷有令，未满五十九岁不许为尼。你若真有此心，不如先做居士，在家修行，也是一条路。”

### 小说/历史素材

- **陈继儒**：1587年焚儒衣冠，隐居东佘山——山人路线的完美原型
- **憨山德清**：万历四大高僧之一，曾因私修寺庙被流放——出家不是避风港
- **《水浒传》鲁智深**：赵员外买度牒让他出家——度牒买卖的文学写照
- **《金瓶梅》寺庙场景**：和尚参与经济活动，做法事赚钱——寺庙经济的真实面

### 交叉互动

- **与支线A（读书人）**：科举落榜→焚儒衣冠→走山人路线——读书人路线的“退出”选项
- **与支线C（织工）**：经营失败→看破红尘→出家/做山人——织工路线的精神出口
- **与支线E（反抗）**：反抗失败→逃入空门——反抗路线的“避难”选项
- **与离乡路线**：山人本质上是“自由人”，最容易选择远游

## 支线E：抗税/反抗路线——“官逼民反”

### 核心定位

不是“选择反抗”，而是“被逼到无路可走”。叙事张力最大的路线，但代价极高——从抗税到请愿到暴力对抗，每一步都在赌命。

### 前置规则：万历年间税监制度

**矿税之祸**：
- 万历二十四年（1596）起，万历皇帝以三大征为借口，派出宦官担任矿监与税使
- 这些人绕过官僚系统，直接对民间搜刮——史称“矿税之祸”
- “通都大邑皆有税监……大珰小监纵横绎骚，吸髓饮血”

**苏州/盛泽的税负**：
- 苏州府田地占全国1%，却负担全国近10%的田亩税
- 还要为皇宫提供丝绸——“半买半上贡”
- 孙隆在苏州设二十多个税卡，税额随意
- 对丝织业的具体加征：每台织机三钱银子，每匹布五分银子，每匹纱三分银子

**对玩家的实际影响**（以3台织机的小机户为例）：

| 税项 | 金额 | 占月收入比 |
|------|------|-----------|
| 织机税 | 3台×3钱=9钱 | 约30%月收入 |
| 布税 | 按产量算，约3-5钱/月 | 约10-17%月收入 |
| **合计** | **约1.2-1.4两/月** | **约40-47%月收入** |

这意味着加税后，小机户几乎一半的收入要交给税监。

### 升级路径：从忍受到反抗

#### 第一级：忍气吞声（交税）

- **状态**：还能勉强交得起，但越来越吃力
- **心理**：希望税监适可而止，相信官府会管
- **选择**：老老实实交税 / 找关系求减免
- **代价**：收入减少，生活质量下降
- **DM引导**：让玩家感受到“温水煮青蛙”——每次加一点，不至于立刻反抗

#### 第二级：消极抵抗（不交或少交）

- **状态**：实在交不起了，或者不想交
- **方式**：拖延不交 / 少报织机数 / 贿赂税吏
- **风险**：税吏上门催征 / 被罚滞纳金 / 被拘押
- **代价**：可能被罚银、被打、被关
- **DM引导**：让玩家发现“不交”的后果比“交”更严重——但“交”也活不下去

#### 第三级：聚众请愿（联名上书）

- **状态**：个体抵抗无效，需要集体行动
- **方式**：联名向知府/巡抚上书 / 集体到税署抗议
- **风险**：被定性为“聚众闹事” / 为首者被抓
- **代价**：为首者可能被杖刑、流放
- **历史参照**：苏州知府朱燮元在织工请愿后宣布撤销新加税收——请愿有时有效

#### 第四级：暴力对抗（民变）

- **状态**：请愿无效，或者税监变本加厉
- **方式**：打死税吏 / 焚烧税卡 / 冲击税署
- **风险**：被定性为“叛乱” / 军队镇压 / 为首者被处死
- **代价**：极高——但叙事张力也最大
- **历史参照**：葛成起义——见下方

### 终极事件：葛成起义（万历二十九年/1601年）

**时间线**：

| 时间 | 事件 |
|------|------|
| 万历二十七年（1599） | 孙隆到苏州，初期“只征行商，不征坐贾” |
| 万历二十九年（1601）夏 | 水灾+丝价暴涨，孙隆加征：每台织机三钱 |
| 六月初 | 机户关门歇业，机工失业 |
| 六月初三 | 葛成在玄妙观聚众，万余织工响应 |
| 六月六日 | 分六队行动，以摇蒲扇为号 |
| 当日 | 打死税棍黄建节、徐怡春，烧毁地痞房屋 |
| 当日 | 孙隆逃往杭州 |
| 随后 | 知府朱燮元宣布撤销新加税收 |
| 事后 | 葛成主动投案，被判死刑后改判 |
| 1601-1613 | 葛成在狱中13年，后获释 |
| 后世 | 苏州人称葛成为“葛贤”“葛将军” |

**起义的纪律性**：
- “不伤无辜，不取金银”——不是暴民，是有组织的请愿
- 只针对税吏和帮凶，不抢不烧普通百姓
- 葛成主动投案——“不能连累大家”

**对玩家的意义**：
- 如果玩家在1587年选择反抗路线，12年后（1599）葛成起义就是终极事件
- 玩家可以选择：参与起义 / 帮助葛成 / 保护家人 / 逃离苏州
- 参与起义的代价极高，但历史证明反抗是有效的——税卡被撤销

### DM植入示例

**场景1：税吏上门**
> 门被拍得山响。你打开门，两个税吏站在外面，后面还跟着几个地痞。“孙公公的新规矩，每台织机加征三钱银子。你有三台，一共九钱，明天之前交齐。”你算了算，这个月卖绸的收入才二两出头，交完税只剩一两——一家人怎么活？

**场景2：丝市上的议论**
> 丝市上没人等活了——机户都关门了。阿根压低声音说：“听说苏州城的机工要闹事，昆山来的葛成在玄妙观聚了几千人。”旁边有人接话：“闹什么闹？打死了税吏，官府能放过你？”又有人说：“可再不闹，大家都得饿死。”

**场景3：做出选择**
> 葛成站在玄妙观的台阶上，手里摇着一把蒲扇——那是行动的信号。几千个织工围在他身边，有人拿着木棍，有人攥着拳头。你站在人群边上，心跳得厉害。加入他们，可能活不了；不加入，也活不下去。你握紧了拳头，往前迈了一步。
