# 松江城市素材：万历年间棉布天下

## 一、城市定位

松江府是全国棉纺织业中心，“衣被天下”不是虚名。万历年间松江府赋税122万两，占全国7%。和盛泽的丝织业完全不同——松江是棉布的天下，“买不尽松江布，收不尽魏塘纱”。玩家从盛泽到松江，面临的是产业转换：丝织手艺不完全适用，需要重新学艺。

## 二、城市面貌

### 2.1 松江府城

- 明末松江大街绵延十余里，“生齿浩繁，居民稠密”的“东南一大都会”（明范濂《云间据目抄》）
- 城市沿市河不断向东西两侧拓展
- 街上布庄比绸庄多，招牌上写的是“标布”“扣布”“三梭布”，不是丝绸
- 空气里飘的是棉花的干燥气息，不是蚕茧的味道

### 2.2 市镇网络

棉纺织业催生了大量市镇：

- **朱泾**：“万家灯火似都城……估客往来多满载，至今人号小临清”
- **朱家角**：“商贾辏集，贸易花、布，京省标客，往来不绝，今为巨镇”
- **枫泾**：布庄上百家，染坊、踹坊应运而生，南北杂货、果品银楼、典当、百货、茶馆、饭店等三百多家
- **七宝**：棉布业市镇中的佼佼者
- 上海县元代只有三个集镇，到明清崛起四十多个，“其形成原因几乎全赖布业”

## 三、棉纺织业细节

### 3.1 全民织布

松江府无论城乡，几乎家家纺纱、户户织布：

- **正德《松江府志》**：“纺织不止乡落，虽城中亦然。里媪晨抱纱入市，易木棉以归；明旦复抱纱以出”
- **“织者率日成一匹，有通宵不寐者”**
- **“农暇之时，所出布匹，日以万计。以织助耕，女红有力焉”**
- 土地大半种植棉花，“海上官民军灶，垦田凡二百万亩，大半种棉，当不止百万亩”

### 3.2 四大布品种

松江棉布发展到明代，形成四大品种七八十种布名：

1. **标布**（大布）：以浦东三林塘所产最精
2. **扣布**（小布/短头布）
3. **稀布**：以龙华、七宝所产为多
4. **高丽布**（洋袍）：出自高桥、洋泾等处

加工后又有括绒布、踏光布、药斑布等名目。松江棉布从最初的本色粗布，发展为可制作衣被、鞋袜、装饰的系列产品。

### 3.3 牙行与布号

- “数百家布号皆在松江、枫泾、朱泾乐业”
- 牙行坐庄收购，也派人到四乡收布，再批发到北方各地
- 外地商人“挟重资而来，白银动以数万两，多则数十万两”
- 《金瓶梅》中都有到松江贩卖棉布的情节
- 顺治十六年（1659），苏松二府布号商人列名碑石者为37家

### 3.4 染坊与踹坊

- 为棉布业服务的染坊、踹坊大批出现
- 与棉布生产有关的鞋店、袜店等专业店应运而生
- 雍正七八年间，松江踹匠1100余人，约为苏州踹匠的十分之一

### 3.5 棉纱船

这是松江最独特的交通细节：

- 万历年间《水陆路程》载：“嘉兴至松江，无货勿顾小船。东栅口搭小船至嘉善县，又搭棉纱船至松江，无虑大船”
- 从嘉善到松江有专门的“棉纱船”，每天将嘉善的棉纱运到松江
- “买不尽松江布，收不尽魏塘纱”——嘉善（魏塘）产纱，松江产布，形成产业链

### 3.6 供应链

- 松江棉布源源不断向北方输出
- 本地棉花已不能满足生产需要
- 北方所产棉花、江浙所产棉纱源源不断输入松江
- 松江成了全国棉纺织业的集散中心

## 四、与盛泽的核心差异

| 维度 | 盛泽 | 松江 |
|------|------|------|
| 产业 | 丝织 | 棉织 |
| 原料 | 蚕桑（本地养蚕） | 棉花（半数田地种棉） |
| 产品 | 丝绸（高端，达官贵人用） | 棉布（大众，天下百姓穿） |
| 利润 | 高但量小 | 薄但量大 |
| 技术 | 织绸 | 织布（技术不同，需重新学） |
| 交易 | 丝绸牙行 | 布号+牙行 |
| 后整理 | 练染 | 染坊+踹坊（踩布） |
| 市场 | 苏州、杭州 | 北方各省、南洋 |

## 五、玩家到达松江的叙事要点

1. **产业冲击**：闻不到蚕茧味，空气里是棉花的干燥气息
2. **技术落差**：丝织手艺不完全适用，织棉布和织丝绸是两套技术
3. **规模震撼**：棉布产量远超丝绸，“日以万计”
4. **转行选择**：是继续做丝织（松江也有少量丝织），还是转做棉布？
5. **供应链视角**：看到从嘉善来的棉纱船，理解产业链的分工
