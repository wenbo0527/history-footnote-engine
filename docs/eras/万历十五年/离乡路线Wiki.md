## 〇、离乡的前提：万历年间的人口流动

**核心事实**：明代中后期，人口流动已是常态。明初的里甲制和关津制（限制人口流动）到万历年间已名存实亡。

- 明中叶后，朝廷放弃了迫使流民返回原籍的努力
- 成化、弘治、正德年间逐步放宽了对逃离户籍所在地者的惩罚
- 一条鞭法推行后，赋役折银，人身控制进一步松弛
- 江南人外出经商、游学、谋生已是普遍现象

**离乡的硬约束**：
1. **路引**：理论上出远门需要官府发的“路引”（通行证），但万历年间执行松懈
2. **户籍**：离开户籍所在地，赋税仍要交——除非你能把户籍迁走
3. **路费**：从盛泽到月港约1000里水路，到北京约2000里运河，都需要不少银子
4. **人脉**：到了陌生地方，没有熟人接应，寸步难行

## 路线一：南下沿海——去月港做海商

### 可行性

**隆庆开关（1567）**后，月港（福建漳州海澄）成为明朝唯一合法的民间海外贸易港口。 到万历十五年（1587），月港已繁荣了20年。

**月港的地位**：
- 被称为“天子之南库”——关税收入巨大
- 通商达47个国家和地区
- 18条航线，开创了“太平洋海上丝绸之路”的黄金时期
- 不足1公里的江岸聚集了7座渡口

### 路线与时间

**盛泽→月港（漳州海澄）**：

| 路段 | 方式 | 时间 | 说明 |
|------|------|------|------|
| 盛泽→杭州 | 水路（江南运河） | 1-2天 | 顺水，方便 |
| 杭州→宁波 | 水路 | 1天 | |
| 宁波→福州 | 沿海船 | 3-5天 | 看风向，可能等风 |
| 福州→泉州→漳州 | 沿海/水路 | 2-3天 | |
| 漳州→月港（海澄） | 九龙江水路 | 半天 | |
| **合计** | | **约8-12天** | 顺利的话 |

### 费用估算

| 项目 | 费用 | 说明 |
|------|------|------|
| 船资 | 约3-5两 | 全程水路+沿海船 |
| 食宿 | 约2-3两 | 沿途10余天 |
| 关卡/税 | 约1-2两 | 各处钞关、厘卡 |
| **合计** | **约6-10两** | 相当于织工2-3个月收入 |

### 到达后的生活

**月港的商机**：
- 丝织品是月港出口的大宗商品——盛泽的丝绸在月港是硬通货
- 可以做“丝客”——从江南收丝，到月港卖给海商
- 也可以直接搭船出海，做海商
- 月港有牙行专门对接海商和内地商人

**月港的风险**：
- 海商竞争激烈，闽南商人垄断大部分生意
- 官府征税越来越重（督饷馆专门收税）
- 海上风险——风浪、海盗
- 去日本是违禁的（隆庆开关规定不得到日本贸易），但私下很多人去

### 女性差异

- 女性可以随丈夫/兄弟南下，但独自出行更困难
- 月港有女性参与贸易的先例——“卖婆”可以随船带货
- 闽南女性有“出海”传统，比江南女性更自由
- 但海上航行对女性更危险——海盗、风浪

> **DM植入示例**：
> - 行会里有人聊：“月港那边，一匹湖绫能卖到三两银子，是这边的好几倍……”
> - 老商人警告：“月港是赚钱，但那边的闽南人抱团，你一个江南人去了，未必吃得开”
> - 牙人暗示：“你要是肯带货下去，我认识月港的陈牙行，可以引荐”

## 路线二：下南洋——出海到东南亚

### 可行性

**隆庆开关后合法**，从月港出发可以合法前往东南亚各地。万历年间，华人下南洋已成规模。

**主要目的地**：

| 目的地 | 航程 | 主要贸易 | 华人社区 |
|--------|------|---------|---------|
| 吕宋（马尼拉） | 约15-20天 | 丝绸换白银（西班牙大帆船贸易） | 已有华人社区 |
| 暹罗（泰国） | 约20-30天 | 丝绸、瓷器换香料 | 已有华人社区 |
| 满剌加（马六甲） | 约30-40天 | 转口贸易 | 已有华人社区 |
| 旧港（巨港） | 约25-35天 | 胡椒贸易 | 已有华人社区 |

**关键背景**：西班牙人在马尼拉经营“大帆船贸易”——把中国丝绸运到美洲换白银。一匹丝绸在马尼拉的售价是进价的3-5倍。 全球白银产量的近一半流入中国。

### 路线与时间

**盛泽→月港→马尼拉**：

| 路段 | 时间 | 说明 |
|------|------|------|
| 盛泽→月港 | 8-12天 | 见路线一 |
| 月港等船/办手续 | 5-15天 | 等季风、办引票 |
| 月港→马尼拉 | 15-20天 | 乘季风，5-7月南风去，10-12月北风回 |
| **合计** | **约30-50天** | 单程 |

### 费用

| 项目 | 费用 | 说明 |
|------|------|------|
| 盛泽→月港 | 6-10两 | 见路线一 |
| 月港引票（出海许可） | 约3-5两 | 官府收费 |
| 船资（月港→马尼拉） | 约5-10两 | 看船型和舱位 |
| 货本 | 量力而行 | 至少10-20两的货才有意义 |
| **合计** | **约25-45两** | 不含货本 |

### 风险

| 风险 | 概率 | 后果 |
|------|------|------|
| 海上风浪 | 高 | 船毁人亡 |
| 海盗 | 中 | 货物被抢，人可能被杀或被卖 |
| 西班牙人征税 | 高 | 马尼拉对华人征重税 |
| 疾病 | 高 | 热带病，无药可治 |
| 回不来 | 中 | 亏本后无钱买回程票，滞留南洋 |

### 女性差异

- 女性下南洋的记录较少，但并非没有
- 闽南有“侨批”传统——丈夫下南洋，妻子在家等“侨批”（汇款+家书）
- 女性独自下南洋极为罕见，但随丈夫同行有先例
- 到达南洋后，华人社区中女性极少，可能面临安全问题

> **DM植入示例**：
> - 茶馆里有人聊：“隔壁张家的二小子去了吕宋，三年寄回来五十两银子……”
> - 老人摇头：“去是去了，十个里头回不来的有三四个，你算算……”
> - 牙人说：“你要是带十匹湖绫下去，到马尼拉至少翻三倍。但船票和引票的钱，你出得起吗？”

## 路线三：走东洋——去日本

### 可行性

**法律上违禁**：隆庆开关明确规定不得到日本贸易。但私下大量商人违禁前往。

**为什么有人冒险**：
- 日本银矿丰富，白银价格低——中国丝绸在日本能换大量白银
- 日本对丝绸需求极大——上层社会以穿中国丝绸为荣
- 利润最高——一匹丝绸在日本可卖到进价的5-10倍

**实际操作**：
- 从月港出发，声称去琉球或吕宋，中途转向日本
- 或从浙江沿海（宁波、双屿）偷渡
- 日本的平户、长崎已有华人社区

### 路线与时间

**盛泽→宁波→平户（日本）**：

| 路段 | 时间 | 说明 |
|------|------|------|
| 盛泽→宁波 | 2-3天 | 水路 |
| 宁波等船 | 5-10天 | 等季风 |
| 宁波→平户 | 5-7天 | 乘季风，春季去 |
| **合计** | **约12-20天** | 比去南洋近得多 |

### 风险

| 风险 | 概率 | 后果 |
|------|------|------|
| 被官府查获 | 中 | 货物没收，可能坐牢 |
| 倭寇/海盗 | 中 | 货物被抢，人被杀 |
| 日本战国末期动荡 | 中 | 万历十五年前后，日本刚统一不久（丰臣秀吉），局势不稳 |
| 回程被查 | 中 | 带白银回来，容易被查 |

### 女性差异

- 女性走东洋几乎不可能独自完成
- 但历史上确有女性被卖到日本的记录（不是自愿的）
- 这条路线对女性玩家来说风险极高，DM应谨慎引导

> **DM植入示例**：
> - 神秘商人低声：“你要是敢走一趟东洋，十匹绸子换回来的银子够你买一栋楼……”
> - 老人警告：“去日本？那是杀头的买卖！上个月刚抓了三个……”
> - 牙人犹豫：“我认识一个船老大，走东洋的，但……你确定？”

## 路线四：北上——走运河去京城

### 可行性

**最常规的远行路线**：京杭大运河是明代南北交通的大动脉，从盛泽出发可以直接走运河到北京。

**北上的理由**：
- 乡试中举后去京城参加会试
- 经商——把丝绸运到北方卖
- 投亲靠友
- 闯荡京城

### 路线与时间

**盛泽→北京**：

| 路段 | 方式 | 时间 | 说明 |
|------|------|------|------|
| 盛泽→苏州 | 水路 | 半天 | |
| 苏州→镇江 | 江南运河 | 1天 | |
| 镇江→扬州 | 渡江+邗沟 | 1天 | |
| 扬州→淮安 | 里运河 | 2天 | |
| 淮安→徐州 | 中运河 | 2-3天 | |
| 徐州→济宁 | 泗水/黄河段 | 3-4天 | 这段最险，黄河段容易出事 |
| 济宁→临清 | 会通河 | 4-5天 | 闸门多，要等 |
| 临清→天津 | 南运河+北运河 | 5-6天 | |
| 天津→北京 | 北运河+通惠河 | 2天 | |
| **合计** | | **约20-25天** | 顺利的话，遇阻可能40天+ |

### 费用

| 项目 | 费用 | 说明 |
|------|------|------|
| 船资 | 约5-8两 | 全程运河船 |
| 食宿 | 约5-8两 | 20余天 |
| 钞关/税 | 约2-3两 | 运河沿线有多个钞关 |
| **合计** | **约12-20两** | |

### 北上后的生活

**京城（北京）**：
- 万历十五年的北京，人口约百万
- 丝绸在京城是好货——宫廷和官员都需要
- 但京城竞争激烈，江南商人已有行会
- 可以投奔同乡会馆（苏州会馆等）

**北方其他城市**：
- 临清：运河大码头，商业繁荣
- 济宁：运河枢纽
- 也可以在运河沿线任何城市下船

### 风险

| 风险 | 概率 | 后果 |
|------|------|------|
| 运河堵船 | 高 | 等几天到几周 |
| 黄河段出事 | 中 | 船翻人亡 |
| 被盗 | 中 | 运河沿线不太平 |
| 北方不适应 | 中 | 气候、饮食、方言都不一样 |
| 京城开销大 | 高 | 房租、吃饭都比江南贵 |

### 女性差异

- 女性走运河北上比走海路安全
- 可以随丈夫/兄弟同行
- 京城有江南来的女眷圈子
- 但独自北上仍然困难——路引、住宿、安全都是问题

> **DM植入示例**：
> - 商人说：“运河上跑一趟，把丝绸运到临清，一匹能多卖五钱银子……”
> - 老人回忆：“我年轻时走过一趟运河，黄河段差点翻船，再也不敢走了……”
> - 秀才说：“明年春闱，我要上京赶考，你要不要搭船同去？”

## 路线五：参军——投军戍边

### 可行性

**万历年间募兵制已全面推行**：卫所制名存实亡，军队主要靠募兵维持。

**募兵的背景**：
- 卫所军户大量逃亡——万历年间“屯田多被势豪侵占，军士无田可耕，逃亡者众”
- 募兵成为主流——戚继光募义乌兵打倭寇是经典案例
- 万历年间有三大征：宁夏之役（1592）、朝鲜之役（1592-1598）、播州之役（1590-1600）
- 1587年（万历十五年）暂时没有大战，但九边（辽东、蓟州、宣府、大同、山西、延绥、宁夏、固原、甘肃）常年需要兵

### 参军的路径

| 路径 | 说明 | 待遇 |
|------|------|------|
| 投九边募兵 | 北方边防，最常见 | 月粮约1-1.5两，但常被克扣 |
| 投京营 | 京城守军 | 月粮约1两，但吃空饷严重 |
| 投地方营兵 | 各省驻军 | 月粮约0.8-1两 |
| 投水师 | 沿海水师，防倭 | 月粮约1两 |

### 参军的风险

| 风险 | 概率 | 后果 |
|------|------|------|
| 军官克扣粮饷 | 极高 | 实际到手可能只有一半 |
| 被当苦力使唤 | 高 | 修城墙、运粮、种地，不是训练 |
| 战死 | 看时期 | 和平期低，战时高 |
| 逃兵被抓 | 中 | 军法处置，可能杀头 |
| 将领私人化 | 高 | 募兵往往只认将不认国，将死兵散 |

**万历十五年前后的大背景**：
- 1587年没有大战，但北方蒙古威胁仍在
- 辽东局势开始紧张——努尔哈赤正在统一女真（1583年起兵）
- 5年后（1592）将爆发朝鲜之役——明朝出兵援朝抗倭

### 女性差异

- 女性不能正式参军（硬约束）
- 但可以随军做“军属”——洗衣、做饭、医护
- 明代有“随营家眷”的传统
- 女性也可以做“女细作”（间谍），但这更偏小说情节
- 更现实的路径：丈夫参军后，妻子在家独自撑起家业

> **DM植入示例**：
> - 招兵的牌子：“九边募兵，月粮一两五钱，愿者速来！”
> - 老兵说：“别信那月粮，到手能有八钱就不错了。但好歹有口饭吃……”
> - 丈夫犹豫：“我要是去投军，你一个人能撑住这个家吗？”

## DM引导原则

1. **离乡不应是“逃避”**——每条路线都有门槛（路费、人脉、勇气），不是想走就能走
2. **离乡后可以回来**——大部分路线是可逆的，玩家可以“出去闯一圈再回来”
3. **离乡改变视角**——出去过的人，回来后看盛泽镇的眼光会不一样
4. **每条路线都是一个新的“时代注脚”**——北上看到朝廷的腐朽，南下看到海洋的广阔，参军看到边防的危机
5. **女性玩家的离乡更难但更有故事**——独自出行的女性，每一步都是挑战，也每一步都是叙事
