## 3. 详细设计

### 3.1 P0-1: 宣纸纹理背景（CSS + SVG 内联噪声）

**问题**：现在 `tokens.css --color-paper: #f5efe1` 是纯色米黄，没有纸张纤维感。

**方案**：
- 不引入外部图片，**全 CSS + 内联 SVG 噪声**
- 替换 `body` 背景 + 主要容器背景

**文件**：[tokens.css](file:///Users/mac/Documents/trae_projects/history_footnote/src/frontend/src/lib/styles/tokens.css) + 新增 `aesthetic.css`

**实现**：

```css
/* src/frontend/src/lib/styles/aesthetic.css */

/* === 宣纸纹理（CSS + 内联 SVG） === */
:root {
  /* 升级 paper 变量为多层背景 */
  --paper-texture:
    /* 第 1 层：极淡径向渐变（模拟纸张透光不均）*/
    radial-gradient(
      ellipse at 30% 20%,
      rgba(255, 248, 230, 0.4) 0%,
      transparent 60%
    ),
    /* 第 2 层：SVG 噪声纹理（宣纸纤维）*/
    url("data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='200' height='200'><filter id='n'><feTurbulence type='fractalNoise' baseFrequency='0.85' numOctaves='3' seed='5'/><feColorMatrix values='0 0 0 0 0.45  0 0 0 0 0.35  0 0 0 0 0.20  0 0 0 0.18 0'/></filter><rect width='200' height='200' filter='url(%23n)'/></svg>"),
    /* 第 3 层：基础米黄 */
    linear-gradient(135deg, #f5efe1 0%, #ede4cc 100%);
  --paper-texture-blend: soft-light, normal, normal;
}

body, .paper-bg {
  background: var(--paper-texture);
  background-blend-mode: var(--paper-texture-blend);
  background-attachment: fixed;  /* 滚动时纹理不移动，模拟"真纸" */
  background-size: 200px 200px, 200px 200px, 100% 100%;
}
```

**关键参数**：
- `baseFrequency=0.85` 纤维粗细
- `numOctaves=3` 层次
- `feColorMatrix` 把噪声染成棕色（0.45/0.35/0.20 模拟纸浆黄）
- `soft-light` 混合模式让纹理与底色自然融合
- `background-attachment: fixed` 滚动时纸张不动（高级感）

**降级**：浏览器不支持 SVG 噪声 → 纯色 fallback（已有）。

### 3.2 P0-2: 朱砂印章（SVG + 点击盖落动画）

**问题**：当前 `Seal.svelte` 组件是简单的 SVG 圆 + 文字，没"盖落"感。

**方案**：
- 重做 `Seal.svelte`：SVG + `feTurbulence` 制造"残缺纹理"（破边、墨不均）
- 点击：`scale(0.92)` + 旋转 `2deg` + 触发 Canvas 墨迹粒子
- 已存在：`<Seal text="..." />` 调用方式不变

**实现**（[`Seal.svelte`](file:///Users/mac/Documents/trae_projects/history_footnote/src/frontend/src/lib/components/design-system/Seal.svelte) 改写）：

```svelte
<script lang="ts">
  interface Props {
    text?: string;          // 印章上的字（默认"注"）
    size?: number;          // px（默认 60）
    rotate?: number;        // 度（默认 -6，模拟手工盖落歪斜）
    onClick?: () => void;
    interactive?: boolean;  // 默认可点击
    inked?: boolean;        // 已被盖过（淡化为淡朱砂）
  }
  let { text = '注', size = 60, rotate = -6, onClick, interactive = true, inked = false }: Props = $props();
</script>

<button
  class="seal"
  class:inked
  style="--size: {size}px; --rotate: {rotate}deg;"
  onclick={onClick}
  disabled={!interactive}
  type="button"
>
  <svg viewBox="0 0 100 100" width={size} height={size}>
    <defs>
      <!-- 残缺纹理：feTurbulence + feDisplacementMap 制造破边 -->
      <filter id="seal-rough">
        <feTurbulence baseFrequency="0.04" numOctaves="2" seed="3"/>
        <feDisplacementMap in="SourceGraphic" scale="3"/>
      </filter>
    </defs>
    <g filter="url(#seal-rough)">
      <rect x="6" y="6" width="88" height="88" rx="3"
            fill="none" stroke="var(--color-cinnabar)" stroke-width="5"/>
      <text x="50" y="68" text-anchor="middle"
            font-family="FZXiaoZhuanTi, 'STKaiti', serif"
            font-size="56" font-weight="900"
            fill="var(--color-cinnabar)">
        {text}
      </text>
    </g>
  </svg>
</button>

<style>
  .seal {
    background: transparent;
    border: none;
    padding: 0;
    cursor: pointer;
    transition: transform 200ms cubic-bezier(0.7, 0, 0.3, 1);
    transform: rotate(var(--rotate));
  }
  .seal:hover {
    transform: rotate(var(--rotate)) scale(1.05);
  }
  .seal:active {
    transform: rotate(var(--rotate)) scale(0.92);
    transition-duration: 100ms;
  }
  .seal.inked {
    opacity: 0.6;
    filter: grayscale(0.3);
  }
  .seal[disabled] {
    cursor: not-allowed;
  }
</style>
```

**关键设计**：
- `feTurbulence + feDisplacementMap`：**SVG 内置滤镜**让边线扭曲，模拟印章的"破边"
- `rotate(-6deg)`：手工盖落歪斜（非完美方正）
- `:active scale(0.92)`：按下时缩一下，松开回弹——像真的在盖印
- `inked` 态：被盖过后变淡（玩家"已确认"信号）

### 3.3 P0-3: 卷轴展开转场（CSS clip-path）

**问题**：当前页面切换是直接淡入淡出，缺少"翻页/展卷"仪式感。

**方案**：路由级转场用 `clip-path: inset(...)` 模拟从中间向两侧展开。

**实现**（[+layout.svelte](file:///Users/mac/Documents/trae_projects/history_footnote/src/frontend/src/routes/+layout.svelte) 新增转场）：

```svelte
<script>
  import { page } from '$app/stores';
</script>

{#key $page.url.pathname}
  <div class="page-transition" in:fly={{ duration: 800, x: -40, opacity: 0 }}>
    <slot />
  </div>
{/key}
```

或更"卷轴"：

```css
.page-transition {
  animation: scrollUnfurl 800ms cubic-bezier(0.4, 0, 0.2, 1);
}

@keyframes scrollUnfurl {
  0% {
    clip-path: inset(0 50% 0 50%);
    opacity: 0;
    transform: scale(0.98);
  }
  60% {
    opacity: 1;
  }
  100% {
    clip-path: inset(0 0 0 0);
    opacity: 1;
    transform: scale(1);
  }
}
```

**注意**：`clip-path: inset(0 50% 0 50%)` 是从中间**合拢**，到 `inset(0 0 0 0)` **展开**——像卷轴从中间向两侧打开。

### 3.4 P1-1: 墨滴扩散粒子（Canvas 2D）

**问题**：当前点击反馈只有 CSS 颜色变化（toast 弹出），没有"墨在纸上晕开"的物理感。

**方案**：新增 `InkBurst.svelte` 组件——挂在游戏页根节点，监听点击，Canvas 渲染粒子。

**文件**：新建 `src/frontend/src/lib/components/effects/InkBurst.svelte`

**实现要点**：

```svelte
<script lang="ts">
  import { onMount, onDestroy } from 'svelte';

  interface Particle {
    x: number; y: number;
    vx: number; vy: number;
    life: number;
    decay: number;
    size: number;
    opacity: number;
  }

  let canvas: HTMLCanvasElement;
  let ctx: CanvasRenderingContext2D;
  let particles: Particle[] = [];
  let rafId: number;
  let dpr = 1;
  let isLowPerf = false;
  let maxParticles = 80;

  onMount(() => {
    dpr = Math.min(window.devicePixelRatio || 1, 2);
    resize();
    window.addEventListener('resize', resize);

    // 帧率检测：低于 30fps 自动降级
    let lastT = performance.now();
    let lowFrames = 0;
    const perfCheck = setInterval(() => {
      const fps = 1000 / (performance.now() - lastT);
      if (fps < 30) {
        lowFrames++;
        if (lowFrames > 3 && !isLowPerf) {
          isLowPerf = true;
          maxParticles = 30;
        }
      } else {
        lowFrames = 0;
      }
      lastT = performance.now();
    }, 1000);

    // 监听所有点击（capture 阶段，确保不会被子元素 stopPropagation 漏掉）
    const onClick = (e: MouseEvent) => {
      const rect = canvas.getBoundingClientRect();
      burst(e.clientX - rect.left, e.clientY - rect.top);
    };
    window.addEventListener('click', onClick, { capture: true });

    loop();

    return () => {
      window.removeEventListener('resize', resize);
      window.removeEventListener('click', onClick, { capture: true });
      clearInterval(perfCheck);
      cancelAnimationFrame(rafId);
    };
  });

  function resize() {
    if (!canvas) return;
    canvas.width = window.innerWidth * dpr;
    canvas.height = window.innerHeight * dpr;
    canvas.style.width = window.innerWidth + 'px';
    canvas.style.height = window.innerHeight + 'px';
    ctx.scale(dpr, dpr);
  }

  function burst(x: number, y: number) {
    const count = isLowPerf ? 20 : 50;
    for (let i = 0; i < count && particles.length < maxParticles; i++) {
      const angle = Math.random() * Math.PI * 2;
      const speed = 0.5 + Math.random() * 1.5;
      particles.push({
        x, y,
        vx: Math.cos(angle) * speed,
        vy: Math.sin(angle) * speed,
        life: 1.0,
        decay: 0.012 + Math.random() * 0.018,
        size: 2 + Math.random() * 4,
        opacity: 0.3 + Math.random() * 0.5,
      });
    }
  }

  function loop() {
    ctx.clearRect(0, 0, canvas.width, canvas.height);
    const ink = getComputedStyle(document.documentElement)
      .getPropertyValue('--color-ink').trim() || '#2c2416';

    particles = particles.filter(p => p.life > 0);
    for (const p of particles) {
      p.x += p.vx;
      p.y += p.vy;
      p.vx *= 0.97;
      p.vy *= 0.97;
      p.life -= p.decay;
      p.size *= 1.008;
      // 软边：径向渐变（不靠 blur，性能更好）
      const grad = ctx.createRadialGradient(p.x, p.y, 0, p.x, p.y, p.size);
      grad.addColorStop(0, `rgba(44, 36, 22, ${p.opacity * p.life})`);
      grad.addColorStop(1, 'rgba(44, 36, 22, 0)');
      ctx.fillStyle = grad;
      ctx.beginPath();
      ctx.arc(p.x, p.y, p.size, 0, Math.PI * 2);
      ctx.fill();
    }
    rafId = requestAnimationFrame(loop);
  }
</script>

<canvas bind:this={canvas} class="ink-burst" aria-hidden="true"></canvas>

<style>
  .ink-burst {
    position: fixed;
    inset: 0;
    pointer-events: none;
    z-index: var(--z-toast);
  }
</style>
```

**使用**：[+layout.svelte](file:///Users/mac/Documents/trae_projects/history_footnote/src/routes/+layout.svelte) 挂一次：
```svelte
<InkBurst />
```

**降级策略**：
- 自动检测帧率 < 30fps → 粒子数 80→30
- `pointer-events: none` 不影响交互
- `prefers-reduced-motion` 媒体查询下完全禁用
- Canvas 失败 fallback 到无效果

## 6. 性能与可访问性

### 6.1 性能

| 关注点 | 方案 |
|---|---|
| 宣纸纹理 | SVG 内联，浏览器一次性缓存 |
| 粒子数 | 单次 50 粒子上限，全局 80 封顶；低帧率自动降到 30 |
| 帧率 | requestAnimationFrame，**自动降级**到 30 粒 |
| dpr | `Math.min(devicePixelRatio, 2)`，避免 4K 屏 4 倍像素 |
| prefers-reduced-motion | 完全禁用粒子 + 转场 |

### 6.2 可访问性

- Canvas 加 `aria-hidden="true"`，屏幕阅读器忽略
- 所有动效通过 `prefers-reduced-motion: reduce` 媒体查询降级
- 印章按钮保留键盘可达（`<button>` 而非 `<div>`）

## 11. 风险与缓解

| 风险 | 概率 | 影响 | 缓解 |
|---|---|---|---|
| 宣纸 SVG 内联太大 | 低 | 网络 | base64 后 ~2KB，可接受 |
| Canvas 粒子在低端机卡顿 | 中 | UX | 帧率检测自动降级 |
| 暗色模式下朱砂对比度不足 | 中 | 可读性 | 用 `--color-cinnabar-light` |
| `prefers-reduced-motion` 用户被忽略 | 低 | A11y | 媒体查询严格降级 |
| 移动端点击事件频繁触发 | 中 | 性能 | 全局 InkBurst 监听用 throttle |
