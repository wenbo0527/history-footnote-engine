# Disco Elysium游戏模式深度调研

## 核心发现

1. **无战斗的RPG**——Disco Elysium是史上第一款将战斗比重压缩到零的CRPG，全部核心玩法由对话+技能检定驱动
2. **技能即性格**——24个技能不是“能力值”，而是“脑海中的声音”，每个技能会主动对你说话、提供选项、影响你的认知
3. **思想内阁**——60种可内化的“思维方式”永久改变角色，每项都是双刃剑（加一个技能、减另一个技能）
4. **检定系统**——骰子+技能+调整值，成功和失败都有独特叙事，失败不是惩罚而是另一种故事
5. **政治性表达**——康米主义、自由市场、保皇派、法西斯四种政治立场贯穿始终，不做简化不做说教

---

## 一、游戏定位：没有战斗的RPG

### 核心设计哲学

Disco Elysium的开发团队ZA/UM来自爱沙尼亚——一个经历过苏联解体、从废墟上重建的国家。主设计师Robert Kurvitz说：

> “我希望能有虚构的宇宙谈论我们自己的现实生活体验。关于我们面对的政治问题，我们周围的地缘政治格局，现代世界的困境。”

游戏设定在瑞瓦肖（Revachol）——一个经历了王室倒台、革命失败、被自由市场联盟控制的废都。玩家扮演一个宿醉醒来、完全失忆的警探，要在调查一桩谋杀案的同时，拼凑自己的过去。

### 为什么没有战斗？

ZA/UM的核心不满：在传统RPG中，战斗系统设计精巧（技能图标花哨、动画酷炫、声效震撼），但**非战斗技能**——辩论、逻辑思维、原创想法、同情心——覆盖了我们95%的日常生活，在游戏设计体系中却只占5%。

> “对话暗藏的疑虑、人心的揣度、我们对信任感的挣扎……思考才是不朽的游戏。”

所以Disco Elysium的第一步设计决策：**一个真正有深度的非战斗技能系统**。

## 二、技能系统：24个“脑海中的声音”

### 四属性×六技能=24个技能

| 属性 | 对应6个技能 | 核心主题 |
|------|-----------|---------|
| **智力** | 逻辑、百科全书、修辞学、概念化、视觉计算、内陆帝国 | 理性与想象力 |
| **精神** | 伏尔泰式幽默、共情、权威、精神力、暗示、内陆帝国 | 情感与直觉 |
| **体格** | 耐力、痛觉耐受、物理器械、电化学、胆量、神之速 | 身体与冲动 |
| **身手** | 感知、反应速度、狡猾、想象力，手眼协调，半影 | 敏捷与洞察 |

### 技能不是数值，是性格

这是Disco Elysium最革命性的设计——**每个技能都是主角脑海中的一个“声音”**：

- **内陆帝国**（高想象力）：会让你对路灯、雨水甚至城市阴影产生哲学对话
- **电化学**（体格下的冲动）：会怂恿你喝酒、嗑药、追求快感
- **权威**：会让你在对话中强硬施压
- **共情**：会让你感受到NPC隐藏的情绪

技能等级越高，这个“声音”越响亮，提供的对话选项越多。**加点不是在强化能力，而是在选择让哪种性格主导你**。

### 技能等级计算

```
最终技能等级 = 初始属性点数 + 习得技能点数 + 专属技能加成 + 道具加成 + 思维加成
```

技能上限受初始属性限制（属性1的技能栈，技能最多2级），但专属技能和思维内阁可以突破上限。

## 三、检定系统：骰子决定命运

### 两种检定

**被动检定**——不由你控制，技能自动触发：
- 走过一面墙，“感知”突然告诉你墙上有弹孔
- 和NPC对话时，“共情”突然提醒你他在撒谎
- 这些是“脑海中的声音”，你无法选择是否听到

**主动检定**——你主动选择使用技能：
- 每段对话至少设有一个主动检定
- 出现在故事的转折点、高潮部分
- 选项描述的是“你试图做的事”，而非“你将会做的事”
- 系统显示成功概率，但结果由骰子决定

### 检定公式

```
检定结果 = 2d6骰子 + 技能等级 + 调整值 vs 难度值
```

**调整值**是关键设计——你在之前对话中获得的信息、完成的任务，都会影响后续检定的成功率。比如：
- 之前把混混的大球扔进海里 → 后续劝说他们放弃三明治时获得调整值加成
- 之前了解了某个NPC的背景 → 后续相关检定更容易成功

### 失败也是故事

**这是Disco Elysium最重要的设计原则之一**——检定失败不是Game Over，而是另一种叙事：

- 成功说服 → NPC配合你
- 失败说服 → NPC拒绝你，但可能触发另一条线索
- 失败开锁 → 你打不开门，但可能从窗户看到里面的东西

> “我们想要每个主动检定的出现都带有强烈的戏剧冲突——不是死胡同就是岔路口的紧要关头。”

### 重试机制

- 白色检定：可以重试（但需要先提升技能或获得新调整值）
- 红色检定：不可重试——一次定终身
- 这种设计让玩家在关键时刻既紧张又兴奋

## 四、思想内阁：内化思维方式

### 机制

思想内阁（Thought Cabinet）是Disco Elysium最独特的角色成长系统：

- 游戏中有**60种思维方式**可以解锁
- 解锁方式：与NPC对话、探索环境、做出特定选择
- 解锁后需要“研究”（花费时间和技能点）
- 研究完成后永久改变角色——**每项都是双刃剑**

### 双刃剑设计

| 思维方式 | 正面效果 | 负面效果 |
|---------|---------|---------|
| 极端自由主义 | 提升相关技能 | 降低其他技能 |
| 内陆帝国 | 增强感知力和想象力 | 削弱争强好胜属性 |
| 第十五个寡头公司 | 获得特殊互动 | 改变对话选项 |
| 慢慢寂寞回家路 | 学习上限提升至5 | 研究期间降低某技能 |

> “思维阁的研究内容普遍呈现出双刃剑效应，每项思维研究大功告成后，往往会提升某一技能数值，同时却降低另一技能数值。”

### 设计意义

思想内阁本质上是在模拟**人的价值观如何塑造行为**——你选择内化什么思想，就决定了你是什么样的人。这与历史注脚体验引擎的“价值观系统”有直接对应关系。

## 五、对话系统：万物皆可对话

### 核心设计

- **没有战斗，只有对话**——所有冲突、探索、成长都通过对话推进
- 每个NPC可能和你聊半小时（真实时间），两三句寒暄就蹦出7个对话分支
- 对话不只是“获取信息”和“推进剧情”，而是**充满心理较劲的互动**

### 对话分支设计

- 对话选项由你的技能组合决定——高“权威”会看到威胁选项，高“共情”会看到安慰选项
- 不同技能组合 = 不同的对话体验 = 不同的故事
- 这意味着**重玩价值极高**——换一种技能组合，看到的是完全不同的世界

### 政治立场对话

游戏中有四种政治立场贯穿对话：

| 立场 | 对应 | 现实原型 |
|------|------|---------|
| 康米主义 | 革命失败后的左翼 | 苏联解体后的东欧左翼 |
| 自由市场 | 联盟控制下的资本主义 | 西方新自由主义 |
| 保皇派 | 怀念旧秩序 | 旧贵族残余 |
| 法西斯 | 极端民族主义 | 种族主义/排外 |

关键：**游戏不做价值判断**——每种立场都有其合理性，也都有其荒谬性。玩家可以选择任何立场，但必须面对其后果。

## 六、与历史注脚体验引擎的可借鉴点

### 直接可借鉴

| Disco Elysium机制 | 历史注脚体验引擎对应 | 借鉴方式 |
|-------------------|-------------------|---------|
| 24技能=24性格 | era.json的价值观系统 | 价值观不只是数值，而是“声音”——DM应该主动用价值观影响叙事 |
| 思想内阁 | 玩家路线选择 | 选择路线后，角色的认知框架永久改变 |
| 被动检定 | DM主动植入 | DM根据玩家属性主动提供信息/线索 |
| 主动检定 | 玩家选择+骰子 | 关键选择有成功概率，失败也是故事 |
| 调整值 | 时代包知识条目 | 之前获得的知识影响后续检定成功率 |
| 政治立场 | 万历十五年的党争/税制 | 不同立场看到不同的世界 |

### 关键设计启示

**1. 失败也是叙事**
- 历史注脚体验引擎应该确保：织户经营失败不是Game Over，而是“织工路线”的入口
- 抗税失败不是惩罚，而是“觉醒路线”的触发器

**2. 技能/价值观是“声音”**
- DM不应该只是被动响应，而应该主动“说话”——用玩家的价值观框架来过滤和呈现信息
- 高“忠君”价值观的玩家，DM应该更频繁地提到“朝廷恩典”
- 高“逐利”价值观的玩家，DM应该更频繁地提到“商机”

**3. 思想/路线内化后永久改变角色**
- 选择“出家路线”后，即使后来还俗，角色的认知框架已经改变
- 选择“抗税路线”后，即使后来妥协，角色对官府的态度已经不同

**4. 政治性表达不做简化**
- 万历十五年的党争、税制、海禁，不应该简化为“好/坏”二元对立
- 每种立场都有其历史合理性和局限性

### 需要注意的差异

| 维度 | Disco Elysium | 历史注脚体验引擎 |
|------|--------------|----------------|
| 主角 | 固定角色（失忆警探） | 玩家自定义（丝织户） |
| 时代 | 虚构世界 | 真实历史（史实不可改） |
| 骰子 | 显式掷骰 | 隐式（DM判断） |
| 文本量 | 百万词级别 | 受LLM上下文限制 |
| 开发模式 | 预写脚本 | LLM实时生成 |
