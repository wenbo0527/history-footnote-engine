### 1) 自动 webp 处理脚本

**位置**：`scripts/regen-icon.sh`

```bash
#!/bin/bash
# 重新处理一个 icon（去白 + 缩放 + webp）
# 用法：./scripts/regen-icon.sh static/icons/stats/cash.jpg
input="$1"
output="${input%.jpg}.webp"
convert "$input" -fuzz 30% -transparent "#FEFDF8" -alpha set PNG32:/tmp/c1.png
convert /tmp/c1.png -bordercolor white -border 1x1 -fuzz 5% -trim +repage \
  -background none -alpha set PNG32:/tmp/c2.png
convert /tmp/c2.png -resize 112x112 -background none -alpha set \
  -gravity center -extent 128x128 PNG32:/tmp/c3.png
cwebp -near_lossless 90 -alpha_q 100 /tmp/c3.png -o "$output"
echo "✅ $output: $(stat -f%z "$output") bytes"
```
