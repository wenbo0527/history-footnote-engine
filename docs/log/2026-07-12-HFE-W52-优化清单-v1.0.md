### P0-1 · 章节蓝图 chapter2-10 缺失（核心 bug · 玩家走到 chapter7 章节化会退出）

- **位置**：`/root/history-footnote-engine/eras/wanli1587/`
- **现状**：仅 `chapter1_blueprint.json` 存在（v1 行）· chapter2-10 完全缺文件
- **信号源**：信号 A5 — 真 LLM smoke ERROR：`第 7 章蓝图加载失败: 蓝图文件不存在: /root/history-footnote-engine/eras/wanli1587/chapter7_blueprint.json，回退硬编码` + 硬编码也没 → "章节化就此退出（无法继续）"
- **影响**：玩家游玩到第 7 章时，章节化叙事体系直接中断 → 玩家卡死 → 必然差评
- **修复方向**：3 选 1
  - **A**：用 minimax-anthropic 跑 1 次生成 chapter2-10 蓝图（≈ 25 万元 token · 等于 5min）· 写入 production eras/ 目录
  - **B**：把蓝图为空时的 fallback 改成"全部用 LLM 实时生成"（代码改动 · 30min）
  - **C**：接受现状（章节 7 必须 LLM 实时生成）= 当前行为 · 但补 LLM fallback 容错（95%）
- **推荐**：**B**（代码修复一次治好）/ 或 A + B 双保险
- **预估修复耗时**：A=10min · B=30min · A+B=40min

### P0-2 · LLM JSON 解析失败 9.7%（章节叙事偶发中断）

- **位置**：`src/history_footnote/chapter/dm_tool.py:LLM JSON 解析` + minimax-anthropic prompt
