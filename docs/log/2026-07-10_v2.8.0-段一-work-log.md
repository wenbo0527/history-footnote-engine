## 🐛 关键问题与解决

### 问题 1：蓝图路径找不到

**症状**：测试 V28_21 失败 `FileNotFoundError: 蓝图文件不存在: /Users/mac/Documents/trae_projects/history_footnote/wanli1587/chapter1_blueprint.json`

**根因**：`root_dir=Path(__file__).parent.parent` 在 tests/ 下解析，但 ChapterFacade 又拼了 `root_dir/era_id` 找路径，**少了一层 `eras/`**

**解决**：
```python
# 改为
@property
def _blueprint_dir(self) -> Path:
    era_id = self.state.era_id or "wanli1587"
    return self.root_dir / "eras" / era_id
```

**教训**：路径拼接要集中在一个属性里，不要散在多个方法。

---

### 问题 2：GameEngineFacade 没有 root_dir 属性

**症状**：测试 V28_26 失败 `'GameEngineFacade' object has no attribute 'root_dir'`

**根因**：`__init__(self, state, era_config, root_dir=None)` 接了参数但**没存为实例属性**

**解决**：
```python
def __init__(self, state, era_config, root_dir=None):
    self.state = state
    self.era_config = era_config
    self.root_dir = root_dir  # 🆕 v2.8.0（ChapterFacade 蓝图加载用）
```

**教训**：构造参数一定要存为实例属性，否则下游用不了。

---

### 问题 3：check_closure() 刚 init 时返回 CONTINUE

**症状**：测试 V28_27 失败 `assert 'CONTINUE' == 'INIT'`

**根因**：`init_chapter(1)` 后 `current_chapter=1`，closure.check() 走 CONTINUE 分支（不是 INIT）

**解决**：
```python
def check_closure(self) -> str:
    cs = self.state.chapter_state
    # 刚初始化那一回合保持 INIT 状态
    if cs.last_closure_status == "INIT" and cs.chapter_start_round == self.state.round_number:
        return "INIT"
    status = self.closure.check()
    cs.last_closure_status = status
    return status
```

**教训**：状态机要有"刚进入"特殊态，不能直接走主流程。

---

### 问题 4：完整生命周期测试期望 16 回合，实际 15 回合

**症状**：测试 V28_41 失败 `assert 15 == 16`

**根因**：节点 4 在 round 13 推进，rounds_in_node = 13 - 12 = 1。round 15 时 rounds_in_node = 3 ≥ 3 → SOFT_READY。**15 回合就完成**。

**解决**：改测试期望为 15。同时发现这暴露了一个**段一未处理的小问题**：16 回合硬收束阈值其实是兜底，正常 SOFT_READY 会在 15 回合触发。

**教训**：写测试前要理解状态机真实行为，**不要用数学直觉（4×4=16）**。
