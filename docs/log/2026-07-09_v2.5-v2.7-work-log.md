|---|---|

| `save_manager` 路径 | 在 `storage/` 下 |

## 🐛 关键问题解决路径

### 问题：玩家看不到命运卡

**症状**：
- 玩家打开 CharCard → 命运卡段为空
- 多次修复但仍不显示

**调查路径**（4 步）：

1. `c4fa961` 改 CharCard 改 UI（**只动了前端**）
2. `584dfa6` 加 format_state 透传（**后端**）
3. `fa8fdbf` 加 mapper 字段（**前端**）
4. `ab3cbf7` 加 session 创建后存档（**持久化**）

**结论**：4 层都断了。`c4fa961` 单独看是 UI 改动，但**没修后端/前端/持久化/重启**。

### 问题：LLM 不可重放

**症状**：
- 同 seed + 同操作 → 输出不同
- LLM temperature 默认 0.7 → 每次采样

**修复**：
- v2.7 加 `make_llm_for_purpose` 工厂
- 4 处调用全部加 `purpose` 参数
- dm/voice 走 0.0，wiki 走 0.3
