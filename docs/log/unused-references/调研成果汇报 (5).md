# 历史注脚体验引擎：玩家离乡城市扩展素材

## 一、扩展思路

当前时代包聚焦盛泽镇，玩家是丝织户。但游戏设计了离乡路线——玩家可能去苏州卖绸、去松江贩棉、去南京赶考、去杭州投亲。每个城市需要**3类素材**：

1. **城市感官层**：玩家到达时看到、听到、闻到什么（DM叙事用）
2. **城市功能层**：在这个城市能做什么（玩家选择用）
3. **城市差异层**：和盛泽有什么不同（认知框架切换用）

---

## 二、从盛泽出发：水路交通

### 2.1 路线与耗时

盛泽位于运河网络枢纽，大运河东线和中线如两条臂膀将其环抱。从盛泽出发，水路可达以下城市：

| 目的地 | 水路路线 | 预估耗时（明代） | 船型 |
|--------|---------|----------------|------|
| **苏州** | 盛泽→平望→运河→枫桥→阊门 | 1天（顺流）~1.5天 | 快船/航船 |
| **杭州** | 盛泽→平望→东线经嘉兴→杭州 | 2~3天 | 夜船（夕发朝至至嘉兴，再半日至杭） |
| **松江** | 盛泽→平望→嘉善→松江 | 2天 | 棉纱船（嘉善至松江有专线） |
| **南京** | 盛泽→苏州→运河→镇江→长江→南京 | 5~7天 | 漕船/客船 |

> 关键细节：明代水路“缓则用游山船慢慢游去，急则夜船可行百里”。从嘉善到松江有专门的“棉纱船”，每天将嘉善棉纱运到松江。

### 2.2 旅途叙事素材

- **船上的时间**：船行缓慢，两岸桑田稻田交替，偶有渔船擦肩而过。船家在船尾煮饭，米香混着河水腥气
- **夜船**：船舱里挤满了客商，有人打鼾，有人低声谈生意。船过闸口时水声轰鸣，船身一震
- **码头**：到苏州阊门码头时，天还没亮，但码头上已经人声鼎沸——挑夫、牙人、船家，各操着不同的口音
- **花费**：盛泽到苏州船资约30~50文，到杭州约80~120文

## 三、苏州：丝织业的中心城

### 3.1 城市感官层

**到达时DM可以这样描述**：

> 阊门码头上，船还没靠稳，岸上的牙人已经扯着嗓子喊起了价。你拎着包袱挤下跳板，脚下是湿漉漉的青石板，头顶是密密麻麻的招牌——“杭绸”“湖丝”“松江棉布”，各色幌子在风里晃。空气里混着桐油、生丝和桂花糕的甜味。一条河，一条街，再一条巷子，人就没了——不是不见了，是被巷子吞了。

**城市特征**：
- 人口约10~40万（学者估计差异大）
- “金阊门银胥门”——阊门是商业中心，胥门是次中心
- 万历年间阊门外山塘街会馆林立：岭南会馆（万历年间建，大运河最早会馆）、潮州会馆、冈州会馆、江西会馆等
- 茶馆星罗棋布，凌晨四点就有老茶客“孵茶馆”
- 观前街是商业核心，玄妙观前商号林立

### 3.2 城市功能层

| 玩家可能做的事 | 具体场景 | 叙事钩子 |
|--------------|---------|---------|
| **卖绸** | 阊门绸庄牙行 | 牙行压价、客商抢货、同行竞争 |
| **买丝** | 湖丝行 | 湖丝价格波动、赊账风险 |
| **赶考** | 府学/贡院 | 科举消息、文人圈动态 |
| **投亲** | 会馆 | 同乡关系、异乡人的孤独 |
| **听消息** | 茶馆 | 朝廷动态、丝价行情、织工传闻 |
| **看病** | 药铺/名医 | 家人生病、医药费 |

### 3.3 城市差异层（vs 盛泽）

| 维度 | 盛泽 | 苏州 |
|------|------|------|
| 规模 | 市镇，万人级 | 府城，数十万人 |
| 节奏 | 慢，熟人社会 | 快，陌生人社会 |
| 丝织 | 家庭作坊为主 | 官营织造局+大作坊 |
| 牙行 | 镇上几个老牙人 | 阊门牙行成行成市，价格更透明但也更残酷 |
| 消息 | 镇上三天传遍 | 当天知道，但真假难辨 |
| 危险 | 乡里人情 | 打行、讼师、骗局 |

## 四、杭州：丝绸的源头城

### 4.1 城市感官层

**到达时DM可以这样描述**：

> 船过拱宸桥，杭州城就铺开了。和苏州不一样，这里的河更宽，桥更高，街上说话的口音也软了几分。清河坊的招牌从街头挂到街尾，丝绸庄的伙计站在门口招呼，一匹杭绸在阳光下闪着水一样的光。你闻到一股桂花香——不是糕点，是巷子里的桂花树。

**城市特征**：
- 南宋旧都，城市格局自南宋定型，“前朝后市、运河贯城、巷陌纵横”
- 清河坊是商业核心，河坊街与南宋御街十字交叉
- 丝绸大作坊：张瀚《松窗梦语》记载的张毅庵，从一张织机发展到二十余张，雇“各色人匠”
- 徐一夔《织工对》记载：工场设在一座老屋中，四五具杼机，十多名工人“手提足蹴”纺织，每天工作到夜二鼓
- 十五奎巷、城隍牌楼一带是市井烟火最浓处

### 4.2 城市功能层

| 玩家可能做的事 | 具体场景 | 叙事钩子 |
|--------------|---------|---------|
| **学艺** | 丝绸大作坊 | 看到规模化生产，震撼于自己的小作坊 |
| **进料** | 杭绸庄/生丝市场 | 杭绸比盛泽绸更精细，价格也更高 |
| **拜佛** | 灵隐寺/净慈寺 | 出家路线的入口 |
| **游湖** | 西湖 | 短暂的逃离感，但银两不够 |
| **投靠** | 杭州织染局 | 官营织造的门槛 |

### 4.3 城市差异层（vs 盛泽）

| 维度 | 盛泽 | 杭州 |
|------|------|------|
| 丝织 | 家庭织户 | 大作坊+官营织染局 |
| 规模 | 镇 | 省城，南宋旧都 |
| 气质 | 市井务实 | 风雅与市井并存 |
| 竞争 | 镇上同行 | 全省丝织中心，高手如云 |
| 机会 | 有限 | 大作坊招工、织染局征匠 |

## 五、松江：棉布的天下

### 5.1 城市感官层

**到达时DM可以这样描述**：

> 松江和盛泽完全不同。这里闻不到蚕茧的味道，空气里飘的是棉花的干燥气息。街上的布庄比绸庄多，招牌上写的是“标布”“扣布”“三梭布”，不是丝绸。里媪们早上抱着纱来市集，换了棉花回去，晚上挑灯再纺——“买不尽松江布，收不尽魏塘纱”，这话你听过，现在亲眼见了。

**城市特征**：
- 全国棉纺织业中心，“衣被天下”
- 万历年间松江府赋税122万两，占全国7%
- “里媪晨抱纱入市，易木棉以归；明旦复抱纱以出”
- 四大布品种：标布（大布）、扣布（小布）、稀布、高丽布
- 外地商人“挟重资而来，白银动以数万两，多则数十万两”
- 松江大街绵延十余里，“生齿浩繁，居民稠密”的“东南一大都会”

### 5.2 城市功能层

| 玩家可能做的事 | 具体场景 | 叙事钩子 |
|--------------|---------|---------|
| **转行** | 从丝织转棉织 | 技术差异、重新学艺的艰辛 |
| **贩布** | 松江布庄 | 棉布生意比丝绸量大但利薄 |
| **买棉** | 棉花市场 | 北方棉花输入松江的供应链 |
| **看染坊** | 染坊/踹坊 | 棉布后整理工序，和丝绸完全不同 |
| **出海** | 松江近海 | 海上贸易路线的入口 |

### 5.3 城市差异层（vs 盛泽）

| 维度 | 盛泽 | 松江 |
|------|------|------|
| 产业 | 丝织 | 棉织 |
| 原料 | 蚕桑（本地） | 棉花（半数田地种棉） |
| 产品 | 丝绸（高端） | 棉布（大众） |
| 利润 | 高但量小 | 薄但量大 |
| 市场 | 达官贵人 | 天下百姓 |
| 转行难度 | — | 丝织手艺不完全适用，需重新学 |

## 六、南京：留都的权力场

### 6.1 城市感官层

**到达时DM可以这样描述**：

> 南京城比苏州大三倍。城墙高得仰头才看得见顶，城门洞里能并排走两辆马车。秦淮河上画舫穿梭，河两岸酒楼茶肆灯火通明——万历年间宵禁已经松了，夜里也能做生意。贡院街不足五百米，却挤满了书肆、笔墨铺和茶馆，空气里飘着墨汁和桂花糕的味道。你一个织户，走在这条街上，觉得自己像一滴水掉进了墨池。

**城市特征**：
- 留都，“天下财赋出于江南，而金陵为其会”
- 人口约40~50万
- 万历中期科举黄金时代，焦竑、朱之蕃、顾起元均列殿试一甲
- 秦淮河夜生活：万历后宵禁松弛，夜间商业社交繁荣
- 江南贡院：科举考场，天下文枢
- 三山街至内桥一带是商业核心，“百货聚焉”
- 《南都繁会图》描绘了南京的繁华盛景

### 6.2 城市功能层

| 玩家可能做的事 | 具体场景 | 叙事钩子 |
|--------------|---------|---------|
| **赶考** | 江南贡院 | 科举路线的核心场景 |
| **告状** | 应天府 | 抗税/维权路线 |
| **拜佛** | 栖霞寺/灵谷寺 | 出家路线 |
| **经商** | 三山街 | 南京市场比苏州更大 |
| **看灯** | 秦淮灯会 | 元宵节叙事事件 |
| **投军** | 南京守备 | 募兵路线 |

### 6.3 城市差异层（vs 盛泽）

| 维度 | 盛泽 | 南京 |
|------|------|------|
| 身份 | 织户 | 无人认识的小人物 |
| 权力 | 镇上无官 | 留都六部、守备、应天府 |
| 机会 | 织绸 | 科举、经商、投军、出家 |
| 危险 | 牙行压价 | 官府、权贵、骗局 |
| 消息 | 镇上传闻 | 天下消息汇聚 |

## 七、era.json扩展建议

### 7.1 新增数据结构

建议在era.json中新增`cities`节点：

```json
{
  "cities": {
    "suzhou": {
      "name": "苏州"，
      "distance_from_shengze": "半日水路"，
      "travel_cost": "30~50文"，
      "sensory": {
        "sight": "阊门码头人声鼎沸，招牌密如林"，
        "sound": "牙人叫价声、船夫号子、茶馆评弹"，
        "smell": "桐油、生丝、桂花糕"
      }，
      "functions": ["sell_silk"， "buy_thread"， "exam"， "guild_hall"， "teahouse"]，
      "key_locations": ["阊门"， "山塘街"， "观前街"， "岭南会馆"]，
      "danger_level": 2，
      "opportunity_level": 3
    }，
    "hangzhou": { ... }，
    "songjiang": { ... }，
    "nanjing": { ... }
  }
}
```

### 7.2 新增触发器

| 触发器 | 条件 | 效果 |
|--------|------|------|
| `arrive_suzhou` | 玩家首次到达苏州 | 切换认知框架→商人视角，解锁苏州茶馆消息源 |
| `arrive_nanjiang` | 玩家首次到达南京 | 切换认知框架→科举/权力视角，解锁贡院消息源 |
| `arrive_songjiang` | 玩家首次到达松江 | 解锁棉织转行路线 |
| `travel_event` | 玩家在水路上 | 随机事件：遇盗/遇客商/遇官船/遇灾民 |

### 7.3 新增认知节点

| 节点 | 解锁条件 | 效果 |
|------|---------|------|
| `城市等级` | 首次离开盛泽 | 理解市镇→府城→省城的层级差异 |
| `棉丝之分` | 到达松江 | 理解丝织和棉织是两个世界 |
| `留都权力` | 到达南京 | 理解权力中心的运作方式 |
| `水路网络` | 首次乘船 | 理解江南水路如何连接城市 |
