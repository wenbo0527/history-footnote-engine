# 苏州城市素材：万历年间丝织中心城

## 一、城市定位

苏州是万历年间江南丝织业的绝对中心。赋税占全国十分之一，民间机户工匠超过三万人，官营织染局工匠约一千五百人。唐寅写阊门：“翠袖三千楼上下，黄金百万水西东。五更市买何曾绝，四远方言总不同。”——这就是玩家到达苏州时应该感受到的冲击：一个比盛泽大十倍、吵十倍、贵十倍的世界。

## 二、街区与地标

### 2.1 阊门——天下第一码头

阊门是苏州西北门，运河在此交汇，形成“五龙汇阊”的水运格局。明清时期被誉为“天下第一码头”。

- **码头**：客货码头众多，天不亮就人声鼎沸。挑夫、牙人、船家各操不同口音
- **钱庄票号**：阊门内钱庄票号林立，银两兑换、汇票往来
- **绸庄牙行**：阊门一带是丝绸交易核心区，牙行成行成市
- **俗语**：“金阊门银胥门”——阊门是苏州最繁华之地

### 2.2 山塘街——七里繁华

山塘街东起阊门渡僧桥，西至虎丘望山桥，长约七里。白居易任苏州刺史时开凿山塘河，河旁筑堤成街。

- **会馆群**：岭南会馆（万历年间建，大运河最早会馆）、潮州会馆、冈州会馆、江西会馆、大兴会馆、东齐会馆、宛陵会馆等
- **商铺**：“居货山积，行云流水，列肆招牌，灿若云锦”（《姑苏繁华图》描绘）
- **曹雪芹评价**：《红楼梦》第一回称阊门山塘一带为“最是红尘中一二等富贵风流之地”
- **河街并行**：水陆并行、河街相邻的格局，建筑精致典雅

### 2.3 观前街——苏州第一商圈

因玄妙观而得名，自明清以来便是商贾云集之地。

- **玄妙观**：西晋咸宁二年（276年）始建，鼎盛时占地五万多平方，近30座殿宇
- **老字号**：陆稿荐（1663年，卤味）、松鹤楼（1737年，松鼠鳜鱼）、黄天源（1821年，糕团）
- **北局**：明清时设有织染局，太监弄因太监驻扎监管织造而得名
- **书场评弹**：光裕公所是苏州评弹行会组织，“千里书声出光裕”

### 2.4 太监弄——从皇家工坊到美食街

- 明永乐年间《苏州织造局志》记载，自洪武三年（1370）起朝廷在苏州设织造局
- 成化年间派太监罗政、陆英、麦秀来苏州监管
- 正德年间派太监龚洪、杨軏等十余人来苏州监管
- 嘉靖年间苏州官营织染局工匠约1500人，民间机户工匠超过三万人
- 太监弄西侧的北局，正是当时的丝绸交易中心

## 三、丝织业细节

### 3.1 机户与机工

万历年间苏州丝织业的核心关系是“机户出资，机工出力”：

- **机户**：拥有织机和机房，出资购买原料，出售丝绸获利。大的有几十张织机，小的也有五六张
- **机工**：无田无产的穷汉，“计日取值”，干一天活发一天工钱
- **劳动力市场**：清晨，机工“仁立桥头”，等待机户雇佣，暮息而归
- **规模**：明末苏州城中有“机户数千人”，按每部织机需三名工人估算，约有一千至两千部织机
- **生存状态**：曹时聘奏报：“吴民生齿最繁，恒产绝少……浮食寄民，朝不谋夕，得业则生，失业则死”

### 3.2 官营织染局

- 苏州织染局是“江南三织造”之一（与江宁、杭州并列）
- 每年通过大运河运往北京的绸缎达二十万匹
- 产品包括妆花缎、织金锦等宫廷专供品种
- 官府以很低价格向本地丝织作坊收购，基本属于半买半上贡

### 3.3 丝绸交易

- 阊门一带牙行坐庄收购丝绸
- 通过大运河运往北方各地
- 苏州上缴税粮几乎占全国十分之一，经济体量巨大

## 四、茶馆与市井

### 4.1 茶馆

苏州茶馆星罗棋布，是信息交换的核心场所：

- **“孵茶馆”**：苏州人泡茶馆的精髓，“孵”字传神——不是喝完就走，是泡在里面
- **凌晨四点**：老茶客天不亮就来，渔民起得早，出门后先来孵茶馆
- **评弹书场**：热闹时围坐台下听评弹、讲书场
- **信息中心**：丝价行情、朝廷动态、织工传闻，茶馆里什么消息都有
- **《万历野获编》记载**：明太祖首创沸水泡茶之法，“开千古茗饮之宗”

### 4.2 市井生活

- **五更市买**：唐寅写“五更市买何曾绝”——天不亮就有早市
- **四远方言**：各地商人汇聚，口音混杂
- **打行**：苏州城内有打行（打手组织），市井暴力
- **讼师**：官司文化发达，讼师替人写状纸

## 五、万历二十九年织工抗税

这是苏州丝织业最重要的历史事件，也是玩家可能亲身经历或听闻的大事：

- 万历二十九年（1601），太监孙隆到苏州加征织造税
- 织工葛成（又名葛贤）率众反抗
- 曹时聘奏报苏州东城百姓家家户户从事纺织业
- 织工抗税的背景：机工“朝不谋夕，得业则生，失业则死”，加税直接威胁生存

## 六、玩家到达苏州的叙事要点

1. **冲击感**：从万人小镇到数十万人的府城，规模差异带来的震撼
2. **陌生感**：熟人社会变成陌生人社会，没有人认识你
3. **机会感**：更大的市场、更多的选择、更高的价格
4. **危险感**：牙行更狡猾、打行更凶狠、骗局更精巧
5. **信息差**：茶馆里的消息比盛泽多十倍，但真假难辨
