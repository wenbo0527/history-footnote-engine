# 杭州城市素材：万历年间丝绸源头城

## 一、城市定位

杭州是南宋旧都，丝绸生产的源头。和苏州不同，杭州的丝织业以大作坊为特色——张毅庵从一张织机发展到二十余张，雇“各色人匠”进行生产。玩家从盛泽到杭州，看到的不是更多的家庭织户，而是规模化的工场生产，这会带来强烈的冲击：原来丝绸可以这样造。

## 二、街区与地标

### 2.1 清河坊——千年商业核心

清河坊是杭州目前唯一保存较完好的旧城区，自南宋以来就是商业核心：

- **南宋御街**：河坊街与中山中路（南宋御街）十字交叉形成核心骨架
- **老字号聚集**：胡庆余堂、方回春堂、叶种德堂、保和堂、张小泉、万隆火腿栈、王星记扇子
- **徐渭对联**：“八百里湖山知是何年图画，十万家烟火尽归此处楼台”——这是对清河坊繁华的真实描绘
- **河坊街**：东西走向，全长约1800米，南宋时是皇城根下的繁华之地

### 2.2 十五奎巷——市井烟火

十五奎巷、城隍牌楼一带是杭州最有烟火气的老街区：

- 清晨菜市场人声鼎沸，居民挑着春笋、河虾、时令蔬菜
- 巷子里的老面馆飘出片儿川的鲜香
- 白墙黑瓦的老房子依巷而建，窄窄的弄堂仅容两人擦肩而过
- 晾衣绳上随风飘动的衣物，构成鲜活生动的市井生活图

### 2.3 巷名里的历史

杭州的巷名藏着半部城市史：

- **孩儿巷**：因陆游诗句千古流传
- **百井坊巷**：源自吴越王凿井惠民
- **马市街、皮市巷、醋坊巷、刀茅巷**：直白记录着古代城市的商业分工
- **岳官巷**：杭州现存规模最大、保存最完整的明代民居——吴宅便坐落于此

### 2.4 运河与码头

- **拱宸桥**：京杭大运河杭州段的标志，船过拱宸桥，杭州城就铺开了
- **武林门码头**：杭州水路客运的起点，可乘船至苏州、湖州等地
- **杭湖线**：杭州至湖州的航线，“一觉困醒到苏州”是杭州人的俗语

## 三、丝织业细节

### 3.1 大作坊制度

杭州丝织业最显著的特征是大作坊：

- **张毅庵**：杭州仁和人，张瀚《松窗梦语》记载，其丝织品“备极精工”，人相争购，从一张织机发展到二十余张，雇“各色人匠”进行生产
- **徐一夔《织工对》**：记载元末明初杭州相安里一个丝织业工场——工场设在一座老屋中，屋内有四五具杼机，十多名工人“手提足蹴”地纺织，每天工作到夜二鼓，“日佣为钱二百缗”
- **官营织染局**：杭州织染局是“江南三织造”之一，与苏州、江宁并列

### 3.2 与盛泽的差异

| 维度 | 盛泽 | 杭州 |
|------|------|------|
| 生产方式 | 家庭织户，自家织机 | 大作坊，雇工生产 |
| 规模 | 一家一两张织机 | 一家可达二十余张 |
| 工人身份 | 自家劳动 | 雇佣工人，“计日受值” |
| 产品定位 | 中等丝绸 | 高端杭绸，“备极精工” |
| 市场 | 镇上牙行 | 全省丝织中心 |

### 3.3 丝绸交易

- 清河坊一带丝绸庄林立
- 杭绸比盛泽绸更精细，价格也更高
- 大作坊的丝绸“商贾所货者，常满户外”——客商排队等货

## 四、南宋旧都的气质

杭州和苏州最大的不同，是南宋旧都的底色：

- **城市格局**：自南宋定型，“前朝后市、运河贯城、巷陌纵横”，至今仍留存古代城市肌理
- **风雅气质**：西湖、灵隐寺、飞来峰，文人墨客的聚集地
- **徐霞客游记**：明崇祯九年（1636），徐霞客在杭州停留三天半，登宝石山“杭城万灶，靡不历历”
- **宗教氛围**：灵隐寺、净慈寺等大寺，出家路线的入口

## 五、玩家到达杭州的叙事要点

1. **震撼感**：大作坊的规模化生产，和盛泽的家庭织户完全不同
2. **风雅感**：南宋旧都的底色，街巷里藏着千年文脉
3. **落差感**：杭绸比盛泽绸精细，自己的手艺在这里可能不够看
4. **选择感**：可以进大作坊做工、可以学新技术、可以拜佛出家
5. **异乡感**：口音不同、规矩不同、物价更贵
