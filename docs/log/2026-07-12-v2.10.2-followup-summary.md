## 💡 关键经验

### 1. 错误信息是金矿

- `'dict' object has no attribute 'current_chapter'` → 100% 暴露 dict/dataclass 不匹配
- `'Page 1 element is not defined'` → 100% 暴露 vite cache 问题
- 用户报 500 → 直接 curl 测每个 endpoint

### 2. 模式化分类

12 个 BUG 根因模式(模式 A-L):
- 模式 A (try/except 吞) - 最严重,33%
- 模式 D (as any) - 最高频,44%
- 模式 B (optional 直渲) - 最易修

### 3. 系统化扫描 > 随机排查

不是 1 个 BUG 1 个 BUG 修,而是**找出根因模式 → 批量预测 → 批量修**:
- 用户报 5 个 → 扫出 8 个预测 → 一并修
- svelte-check 22 个 → 一起修
- 前端旧代码 8 类 → 写审计报告 + 清理

### 4. SSR vs CSR 边界

`+page.svelte` 调 `$game` store,SSR 时 $game 为 null → 500
→ 强制 `ssr = false`(纯 client-only)
→ 永久避免 SSR 状态相关 BUG
